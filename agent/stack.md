# Tech Stack

## Frontend

- Next.js (App Router)
- Plain CSS (no heavy UI library)
- Single page

## Backend

- Next.js API Routes (Node runtime)

## Database

- PostgreSQL

## ORM / Query

- Drizzle ORM (recommended)
  or
- Prisma (acceptable)

## Hosting

- Frontend + API: Vercel
- Database: Supabase / Neon

## Why

- Fast setup
- Minimal infra overhead
- Easy deployment
